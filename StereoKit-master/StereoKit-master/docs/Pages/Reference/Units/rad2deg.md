# [Units]({{site.url}}/Pages/Reference/Units.html).rad2deg

<div class='signature' markdown='1'>
static float rad2deg
</div>

## Description
Radians to degrees, multiply radian values by this, and you get
degrees! Like so: `PI * Units.rad2deg == 180`
